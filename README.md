# The Signature Music: Events & Financial Analysis

## 📌 Overview
This repository showcases a real-world BI application for "The Signature Music." The dashboard serves as a central hub for tracking event profitability, managing artist bookings, and monitoring banking transactions, replacing fragmented spreadsheets with a "Single Source of Truth."

## 🛠️ Tech Stack
* **Tool:** Power BI
* **Domain:** Event Management & Financial Operations
* **Features:** Financial reconciliation, vendor payout tracking, and event lifecycle visualization.

## 📈 Business Impact
* **Financial Clarity:** Automated the calculation of net profit per event by aggregating multi-source transaction data.
* **Operational Efficiency:** Streamlined artist management by visualizing communication statuses and booking timelines.
* **Audit Readiness:** Provided a structured digital trail for all event-related expenditures and revenues.
